## Chatroom
- *Chatroom* 是一个基于网页的在线聊天应用。这是一个实训练手项目。

#### 0.命名规范提示
__A__ 小写下划线
__B__ 首字母大写驼峰
__C__ 首字母小写驼峰

1. 所有`folder`__A__ `./components/navigation`.
2. `*.vue`:
   - `/views`:  __A__  `profile.vue`.
   - `/components`：__B__  `Profile.vue`.
3. `*.js`: __A__  `index.js`.
